# Employee Management System

## Overview
A console-based Employee Management System developed in Java as part of the Maincraft Technology Java Developer Internship Task 1.

## Features
- Add Employee
- View Employees
- Search Employee by ID
- Update Employee Department
- Exit Application

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner Class

## Project Structure
Employee.java
EmployeeApp.java

## How to Run
1. Compile the files:
   javac Employee.java EmployeeApp.java

2. Run the application:
   java EmployeeApp

## Sample Operations
- Add employee records
- View all employees
- Search employees by ID
- Update employee department
